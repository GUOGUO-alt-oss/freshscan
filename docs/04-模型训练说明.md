## 1. 数据集

### 1.1 数据来源

| 项目 | 内容 |
|------|------|
| 名称 | Fruits Fresh and Rotten for Classification |
| 地址 | <https://www.kaggle.com/datasets/sriramr/fruits-fresh-and-rotten-for-classification> |
| 总样本数 | 13,599 张图片 |
| 类别数 | 3 种水果 × 2 状态（新鲜/腐烂）= 6 类 |
| 图片格式 | JPEG, 尺寸不一（100×100 ~ 512×512） |

### 1.2 类别列表

| 索引 | 标签名 | 中文名 | 状态 |
|------|--------|--------|------|
| 0 | `fresh_apple` | 苹果-新鲜 | Fresh |
| 1 | `rotten_apple` | 苹果-腐烂 | Rotten |
| 2 | `fresh_banana` | 香蕉-新鲜 | Fresh |
| 3 | `rotten_banana` | 香蕉-腐烂 | Rotten |
| 4 | `fresh_orange` | 橙子-新鲜 | Fresh |
| 5 | `rotten_orange` | 橙子-腐烂 | Rotten |

> **注**：芒果(mango)、草莓(strawberry)、番茄(tomato) 三个类别在实现中已经被排除，但原始数据集仍包含。

### 1.3 数据划分

| 子集 | 比例 | 用途 |
|------|------|------|
| 训练集 (train) | 80% (~10,880 张) | 模型训练 |
| 验证集 (val) | 10% (~1,360 张) | 超参调优、早停判定 |
| 测试集 (test) | 10% (~1,360 张) | 最终评估 |

划分策略：**分层抽样（stratified split）**，保证每个子集中 6 个类别的比例与原数据集一致。

## 2. 模型架构

### 2.1 基线模型：MobileNetV3-Small

| 属性 | 值 |
|------|-----|
| 基础模型 | MobileNetV3-Small |
| 预训练权重 | ImageNet (imagenet) |
| 输入尺寸 | 224 × 224 × 3 (RGB) |
| 输入值范围 | [0, 1] (float32) |
| 归一化方式 | 像素值 `/ 255.0`（mean=[0,0,0], std=[255,255,255]） |
| 分类头 | GlobalAveragePooling2D → Dropout(0.2) → Dense(6, raw logits) |
| 可训练参数 | ~2.5M（大部分冻结，仅分类头 + 最后几层可训练） |

### 2.2 备选方案：ResNet-18

若 MobileNetV3-Small 在测试集上 top-1 accuracy < 85%，升级为 ResNet-18（ImageNet 预训练）作为备选。ResNet-18 参数量约 11M，推理延迟预计 80-150ms（CPU），需额外评估 APK 增量。

## 4. 训练步骤

### 4.1 代码骨架

```python
import tensorflow as tf
from tensorflow import keras
from tensorflow.keras import layers

# 1. 加载预训练模型（不含分类头）
base_model = keras.applications.MobileNetV3Small(
    input_shape=(224, 224, 3),
    include_top=False,
    weights='imagenet',
    include_preprocessing=False  # 自行归一化
)

# 2. 冻结基础层（仅训练最后 20 层）
base_model.trainable = True
for layer in base_model.layers[:-20]:
    layer.trainable = False

# 3. 构建分类头
inputs = keras.Input(shape=(224, 224, 3))
x = base_model(inputs, training=False)
x = layers.GlobalAveragePooling2D()(x)
x = layers.Dropout(0.2)(x)
outputs = layers.Dense(6, activation=None)(x)  # raw logits
model = keras.Model(inputs, outputs)

# 4. 编译
model.compile(
    optimizer=keras.optimizers.Adam(learning_rate=1e-4),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)

# 5. 训练
callbacks = [
    keras.callbacks.EarlyStopping(patience=5, restore_best_weights=True),
    keras.callbacks.ReduceLROnPlateau(factor=0.5, patience=3),
    keras.callbacks.ModelCheckpoint('best_model.keras', save_best_only=True),
]

history = model.fit(
    train_generator,
    validation_data=val_generator,
    epochs=50,
    callbacks=callbacks
)
```

### 4.2 评估

```python
# 在测试集上评估
```

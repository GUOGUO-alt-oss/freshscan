# 「鲜识 v2.0」UI 设计规格文档

> 版本：v2.4 | 日期：2026-06-17 | 关联：01-PRD-v2 / 02-架构设计-v2

---

## 1. 设计原则

| 原则 | 说明 |
|------|------|
| **一页到家** | 首页展示全部功能入口，零菜单深度 |
| **拍完即得** | 拍照→自动分析→结果弹出，中间无任何用户操作 |
| **小白友好** | 菜谱步骤 ≤ 5 步，筛选代替搜索，大按钮 + 大数字 |
| **M3 原生** | Material Design 3 全部组件，28dp 大圆角，Android 12+ 动态取色 |
| **减脂心智** | 绿色 = 低热量（< 200kcal），橙色 = 中等（200-400），红色 = 高（> 400） |

---

## 2. 颜色系统

| 角色 | 色值 | 用途 |
|------|------|------|
| Primary | `#2E7D32` | 主按钮、选中 Chip、标题栏 |
| Primary Container | `#E8F5E9` | 减脂 Chip 选中背景 |
| Secondary | `#00897B` | 步骤进行中、次要强调 |
| Error | `#F44336` | 腐烂标识、高热量标签 |
| Surface | `#FFFFFF` / `#1E1E1E` | 卡片、页面背景 |
| Outline | `#E0E0E0` | 卡片边框、分隔线 |
| On Surface Variant | `#6B7280` | 次要文字 |
| 粒子主色 | `#FFFFFF` alpha 0.8 | 分析页粒子 |
| 粒子辅色 | `#FFD54F` alpha 0.4 | 粒子金色点缀 |

**BottomSheet 状态切换：**
- `COLLAPSED`（Phase 2 初始）→ 220dp 高，显示食材列表
- `HALF`（上滑一次）→ 450dp 高，食材 + 筛选 Chip
- `FULL`（上滑到底）→ 全屏，食材 + Chip + 菜谱 LazyColumn

**手势冲突处理：** LazyColumn 内容未到顶部时拦截 BottomSheet 拖拽；内容到达顶部后 BottomSheet 可正常拖拽

### 4.3 错误状态

| 状态 | 表现 |
|------|------|
| 检测无结果 | 粒子动画结束 → BottomSheet 显示 "未检测到果蔬" + 空状态图标 + [重新拍照] FilledButton |
| 全水果 (无可烹饪) | 食材列表正常显示 → 底部文字 "检测到的全是水果，建议直接食用 🍎" + 无[找菜谱]按钮 |
| 菜谱匹配空 | 筛选 Chip 下显示 "未找到匹配菜谱，试试其他分类" |
| 模型加载失败 | 粒子动画 500ms 后 → Snackbar "AI 引擎启动失败" + [重试] 按钮 |

## 5. 动画规格

| 动画 | 触发 | 技术 | 时长 |
|------|------|------|------|
| 粒子扫描 | 分析页进入 | Canvas + `animateFloatAsState` + 200 粒子随机运动 | 2s |
| 暗色遮罩渐入 | 同上 | `animateFloatAsState(alpha: 0→0.4)` | 500ms |
| BottomSheet 滑入 | 粒子动画结束 | M3 `ModalBottomSheet` 默认动画 | 300ms |
| Chip 选中 | 点击 FilterChip | M3 默认 `animateColorAsState` | 300ms |
| 菜谱卡片入场 | LazyColumn scroll | `AnimatedVisibility(fadeIn + slideInVertically)` 交错延迟 50ms | 300ms |
| 步骤切换 | 点击下一步 | `animateContentSize` + 颜色 灰→绿 | 200ms |

## 7. 菜谱图片资源方案（v2.4 新增）

### 7.1 资源规划

菜谱 Hero 图 + 卡片缩略图共需 ~110 张图片（100+ 菜谱 + 预留）。

| 方案 | 体积 | 维护 | 采用 |
|------|------|------|------|
| APK assets (WebP, 80dp 缩略图) | ~2.5MB | 人工制作 | ✅ 推荐 |
| 纯色背景 + emoji 占位 | 0MB | 无 | 备选降级 |

### 7.2 规格

- 格式: WebP (lossy, quality 80%)
- 尺寸: 卡片缩略图 160×160px (80dp@2x), Hero 大图 800×400px (200dp@2x)
- 存储: 
- 加载: Coil  指定 

### 7.3 Recipe JSON 扩展字段

